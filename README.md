# Pill-Dispenser
An Automatic Pill Dispenser for Arduino
## Features
* 

## Components
* Arduino Uno r3
* Display LCD 2004 + i2c Module
* Stepper motor 28byj-48 5v + Driver uln2003
* RTC module ds3231
* Passive buzzer module mh-fmd
* MB102 Breadboard Power Supply Module 3.3V 5V
* Power Supply 12v 2A
